# Zomato-Sales-Dashboard-Power-BI-Project
This project presents an end-to-end Power BI dashboard built to analyze Zomato sales data and uncover insights related to order trends, customer behavior, city-wise performance, and cuisine preferences. The dashboard is designed to support data-driven decision making through interactive and visually intuitive analytics.

# Dataset
500 order records
Multiple cities, cuisines, restaurants, and customers
Fields include order value, order date, city, cuisine, payment mode, order status, and delivery ratings
Tools & Technologies

# Power BI Desktop
Power Query for data cleaning and transformation
DAX for calculated measures and KPIs
Data Processing
Cleaned and transformed raw data using Power Query
Created date attributes (Year, Month Name, Month Number)
Applied proper month sorting using numeric month values
Built reusable DAX measures for sales, orders, customers, and averages

# Dashboard Features
KPI cards for Total Sales, Total Orders, Average Order Value, and Total Customers
Monthly sales trend analysis
City-wise sales performance
Cuisine-wise order distribution
Interactive slicers for Year, City, and Cuisine

# Key Insights
Metro cities contribute the majority of total sales
Certain cuisines consistently dominate order volume
Order trends vary significantly across months and locations
High delivery success rate indicates reliable service performance

# Outcome
The project demonstrates practical skills in data modeling, DAX, and dashboard design, making it suitable for academic submission, portfolio showcasing, and entry-level data analyst roles.
